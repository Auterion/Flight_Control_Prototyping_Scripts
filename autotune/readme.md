# Installation using a virtual environment
## Create a new venv
```
python3.9 -m venv virtualenv-test
source virtualenv-test/bin/activate
```

## Install the dependencies
```
pip3 install scipy pyulog control pyqt5
```

## Start the GUI
```
python3 autotune.py
```
